# WebCrawler 🕷️

WebCrawler para verificar os papéis da bolsa de valores e ordena os 10 primeiros de acordo com os valores de mercado:

- www.fundamentus.com.br

## Running:

- make build
- make start

## Stopping the containers:

- make stop

\*Make sure you have docker installed
